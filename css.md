# Bootstrap 4, Foundation 6, dan Tailwind CSS

## 1. Bootstrap 4
```
Bootstrap dikembangkan di Twitter sebagai alat internal oleh dua karyawan mereka Mark Otto dan Jacob Thornton untuk menghapus inkonsistensi antara pengembang. Kemudian mereka memutuskan untuk merilisnya sebagai open source pada 19 Agustus 2011. Kemudian namanya Twitter diturunkan dari Twitter Bootstrap. Versi saat ini dari kerangka adalah V4.0.0 beta 1.
```
* Popularitas
```
Bootstrap tidak diragukan lagi yang paling populer di antara semua front end framework. Versi terbaru Bootstrap 4 lebih baik daripada versi sebelumnya. Bootstrap adalah project paling populer di GitHub.
Ketika membandingkan Bootstrap dan Foundation, Bootstrap di nomor jauh lebih unggul dari foundation. Pada saat penulisan, Bootstrap memiliki 115,193 bintang-bintang di GitHub dan Foundation memiliki 22,227 bintang-bintang.
```
* Komponen
```
Bootstrap dan Foundation memfasilitasi coding dan membantu kita menghemat waktu dengan menggunakan komponen pra-dibangun yang kami dapat membangun menjadi aplikasi kita. Pada pandangan pertama, keduanya memiliki komponen yang serupa dengan desain yang berbeda. Tetapi jika kita mengambil melihat lebih dekat, permukaan perbedaan.
Dalam persyaratan Desain, komponen Bootstrap tampaknya menjadi lebih siap digunakan. Ada lebih banyak contoh untuk masing-masing komponen di Bootstrap daripada di Yayasan.
Keduanya kerangka kerja memiliki komponen tertentu, seperti tombol, peringatan, bentuk, tab dll tetapi mereka berbeda dalam komponen seperti komidi putar, scrollspy Jumbotron, popovers dll. Lalu ada daftar inline, harga tabel dan video dalam Yayasan.
```
* Dokumentasi
```
Bootstrap versi ini telah menulis ulang semua dokumentasi dan telah menambahkan plugin. Artinya dengan begitu kita akan dipermudah dalam menggunakan kode-kode Bootstrap.
```
* Terdapat pilihan Kustomisasi
```
Dengan adanya fitur ini kita dapat melakuakn kustomisasi pada elemen class yang tersedia, sebagai contoh kita bisa menghilagkan atau mengubah corner, shadow, gradient dan elemen class lain-lain dengan cara mengupdate variable dan melakukan kompilasi, dengan cara tersebut kita tidak lagi menggunakan banyak file stylesheet.
```
* Peningkatan pada grid sistem
```
Pada Bootstrap 4 grid sistem telah ditingkatkan, dengan adanya peningkatan ini akan meningkatkan pula pada target perangkat mobile yang lebih baik.
```
* Dukungan IE8 dihapus
```
Dengan penghapusan dukungan IE8 ini maka Boostsrap telah menggantinya dengan dukungan unti rem dan em. Tujuan nya untuk meminimalisir dan menghindari dari peretasan lewat CSS.
```
* HTML reset telah diubah ke modul terbaru yakni Reboot
```
Hal ini bertujuan untuk kita dapat menimpa pengaturan CSS lebih baik tanpa adanya penggunaan !important.
```
* Dihapusnya Glyphicons
```
Pada Bootstrap 4 Glyphicons telah dihapus dari build. Dengan penghapusan Glyphicon Bootstrap 4 hanya memberikan arahan atau petunjuk untuk menggunakan paket icon pihak ketiga.
Selain dari fitur-fitur diatas, Bootstrap 4 masih banyak lagi memberikan fitur-fitur dan komponen terbarunya.
Catatan: Bootstrap tidak lagi memberikan support pada versi 2.x, tetapi masih berlaku untuk versi 3.
Sekian info mengenai Rilis Bootstrap 4 dan Fitur Terbarunya. Semoga dapat bermanfaat.
```
